# SmartTravel UK

## Description

這是一個結合金融分析邏輯與英國鐵路數據的智能預測平台。

## Core Strategies (核心策略)

### 地緣政治與板塊聯動 (The Geopolitical Overlay)

分析營運商（TOCs）之間的關聯性與外部事件溢出效應。

### 軋空潛力 (Short Squeeze)

監控 Advance Ticket 剩餘量，捕捉票價跳升前的買入點。

### IV 偏斜與波動 (Volatility Dynamics)

捕捉因節日或賽事導致的非理性票價波動。

### 現金跑道 (Cash Runway)

嚴格控管 RDM API 呼叫成本與營運風險。

### RSI 指標背離 (RSI Divergence)

利用動能指標判斷票價超買/超賣的反轉黃金時機。

## Technology Stack

HTML5, JavaScript, GitHub Pages, and Rail Data Marketplace (RDM) API integration.

## Multi-language Support

支持中英文動態切換。
